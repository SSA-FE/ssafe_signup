## ⚙ 기능
-  Home 페이지에서 회원가입 버튼 누르면 SignUp 페이지로 이동
-  이메일, 비밀번호, 비밀번호 확인 유효성 검사
-  회원가입 완료 시, SignUp 페이지에 입력받은 데이터 알림창 등장
-  알림창 확인 버튼 누른 후, Home 페이지로 다시 이동
